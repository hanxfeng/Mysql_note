- # MYSQL的启动：
	- 在任务管理器最下方的服务中找到MYSQL右键即可启动或停止，或在cmd中输入net start/stop mysql80（此处输入版本号，我的为80）


- # 进入MYSQL：
	- 在cmd中输入mysql -h ip -p 端口号 -u root -p 密码 如果是在本地，-h ip可以省略，端口号未改动可以只输入-p。或在mysql.exe进入

- # 注释方式：
	- 单行--或#，多行/* 注释 */


- # sql语句的分类：
	- ## DDL：
		- 数据定义语言，用于定义数据库对象（数据库，表，字段）
	
	- ## DML：
		- 数据操作语言，用于对数据库对象进行增删改
	
	- ## DQL：
		- 数据查询语言，用于查询数据库中表的记录
	
	- ## DCL：
		- 数据控制语言，语言创建数据库用户，控制数据库访问权限
