- 字符串和日期数据应包含在引号中


- # insert 添加数据：

	- ## 给指定字段添加数据：
	  
		- `insert into 表名 (字段1，字段2) values (值1，值2);`
		
		- 值的数量要与字段的数量相同，字段名无需加引号
	
	- ## 给全部字段添加数据：
	  
		- `insert into 表名 values (值1，值2，值3);`
		
		- 添加的值的数量要与表中所含字段的数量相同
	
	- ## 批量添加数据：
	
		- `insert into 表名 (字段1，字段2) values (值1，值2),(值1，值2),(值1，值2);`
		
		- `insert into 表名 values (值1，值2，值3), (值1，值2，值3), (值1，值2，值3);`
		
		- 添加的值的数量要与表中所含字段的数量相同
	
	- ## 注意事项：
	
		- 插入时，字段顺序与值的顺序一一对应
		
		- 字符串和日期数据应该包含在引号中
		
		- 插入数据大小应在字段范围内，例如不能给varchar（10）传入一个长15的值


- # update 修改数据：

	- ## 修改数据：
	
		- `update 表名 set 字段名1=值1,字段名2=值2 [where 条件];`
		
		- 条件处为筛选条件，例如可以筛选另一个字段中值为1或大于1的数据修改
	
	- ## delete 删除数据：
	
		+ `delete from 表名 [where 条件];`
		
		- 条件处为筛选条件，具体同update
		
		- delete只能删除整行，不能删除某一个变量（字段），需要删除应使用update替换为null
		
		- 条件处相等为=而非py中的==
