函数：

字符串函数： 在单独使用时需要在前面加上select

字符串拼接：concat(S1,S2,….Sn)

将字符串S1，S2……Sn拼接为一个字符串

字符串大小写转变：

将字符串str转为小写：lower（str）

将字符串str转为大写：upper（str）

左填充：lpad（str，n，pad）

用字符串pad对字符串str进行左填充，直到达到n个字符串的长度

右填充：rpad（str，n，pad）

用字符串pad对字符串str进行右填充，直到达到n个字符串的长度

trim（str）

去掉字符串首尾的空格，中间的空格不会被去除

substring（str，start，n）

返回字符串从start位置起的n个长度的字符串，sql中下标从1开始

数值函数：

ceil（x）

对x向上取整

floor（x）

对x向下取整

mod（x，y）

返回x/y的余数

rand（）

返回0到1之间的随机数

round（x，y）

对x进行四舍五入，保留y位小数

日期函数：

curdate（）

返回当前日期，格式为Y-M-D

curtime（）

返回当前时间，格式为H:M:S

now（）

返回当前时间和日期，格式为Y-M-D H:M:S

year（date），month（date），day（date）

获取指定日期date的年，月，日

date_add（date，INTERVAL expr type）

返回一个日期/时间，加上expr后的结果，type是expr的类型，如year，hour等

如果要对时间进行操作，必须要加上日期
INTERVAL 可以理解为是参数名

datediff（date1，date2）

返回date1和date2之间相差的天数，结果可以是负数,单位是天

流程函数：

if（value，t，f）

如果value（通常是一个比较运算符）的值为True，则返回t，否则返回f

ifnull（value1，value2）

如果value1的值不为null，则返回value1，否则返回value2

Case

基本语法为case value when a then b (when a2 then b2) else c end

其中value是要进行判断的值，when相当于py中的if，then相当于if中的语句，大意是如果value的值为a，那么改为b，否则为c，a也可以为比较运算符，when then可以出现多个，但必须成套出现
在使用一个大于85一个大于60时可以只写大于60和大于85,不用额外写小于85
