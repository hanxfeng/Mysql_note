介绍：
	存储过程是1事先经过编译并存储在数据库中的一段MySQL语句集合，调用存储过程可以减少开发人员的很多工作，减少数据在数据库和应用服务器之间的传输，提高数据处理效率。存储过程思想上就是对sql语句进行封装与重用，类似于函数，以下非代码均以函数代替
特点：
	1.封装复用
	2.可以接收参数也可以返回数据
	3.减少网络交互，提升访问效率
基本语法：
	创建：
```
		creat procedure 函数名（参数)
		begin
			sql语句
		end;
```
调用:
	`call 函数名（参数）`
查看：
	1.查询指定数据库的存储状态及状态信息
		`select * from information_schema.routines where routien_schema = 数据库名`
	2.查询指定函数的创建语句
		`show creat procedure 函数名`
删除：
	`drop procedure [if exists] 函数名`
		if exists：如果存在则删除
额外提示：
	在命令行中创建函数时，需要使用`delimiter 新结束符`来指定新的结束符，不然以分号作为结尾会使代码在SQL语句部分中断
系统变量：
	MySQL服务器提供的变量，分为全局变量与会话变量：
		全局变量(global)：
			在所有会话中都有效的变量
		会话变量(session)：
			仅在创建它的会话中有效的变量
	查看系统变量：
		`show [global | session] variables;`   查看所有系统变量
		`show [global | session] variables like '...'`  使用模糊匹配查看系统变量
		`select @@[global | session].系统变量名;` 或  `select @@变量名` 查看指定变量的值
		[global | session] 可以不进行指定，默认是 session
	设置系统变量：
		`set [global | session] 系统变量名=值;`
		`set @@[global | session]系统变量名=值;`
		设置的系统变量修改在服务重启之后会失效。
自定义变量：
	定义：
		用户根据自身需求自定义的变量，使用时直接 @变量名 使用即可，作用域为当前会话
	赋值：
		`set @变量名 =/:= 值 [,@变量名2=值2];` 可以一句中同时为多个变量赋值
		`select * into @变量名 from 表名;` 将查询结果赋值给某个变量
		对未赋值的变量进行使用，这个变量的值为null，不会报错
局部变量：
	定义：
		局部变量是根据需要在局部生效的变量，仅在类似函数，存在begin end的地方使用，使用前需要使用declare声明，可作为作为储存过程内的局部变量和输入参数，局部变量生效范围在声明的begin end 之间。
	声明：
		`declare 变量名 变量类型 [default 默认值]` default用于指定默认值
		赋值方法与自定义变量相同，但不用在变量名前加@

参数：
	类型：
		in：
			输入参数，调用时需传入参数值，默认类型
		out：
			输出参数，作为返回值
		inout：
			即可做输入值，也可做返回值，一般需要传入一个值操作后传输使用
	使用：
		`creat procedure 函数名(参数类型 参数名 参数类型，参数类型 参数名2 参数类型)`
		第一个参数类型指输入参数，输出参数，第二个参数类型指int,double等
		不需要像python指定输出
		示例1：
```
			create procedure p1(in score int,out result varchar(10))
			begin
				if score >= 60 then
					set result = '及格';
				elseif score >= 30 then
					set result = '勉强';
				else
					set result = '不及格';
				end if;
			end;
```
		示例2：
```
			creat procedure p2(inout score double)
			begin
				set score = score/2;
			end;
```

游标 cursor
	定义：
		游标是用来存储查询结果集（多行多列数据）的数据类型，在存储过程和函数中可以使用游标对结果集进行循环处理。
	语法：
```
		%声明，查询语句指游标存储的是哪个表查询出的结果集，先声明变量，再声明游标
		declare 游标名 cursor for 查询语句;
		%打开游标
		open 游标名;
		%获取游标记录，一般配合循环使用，fetch在循环中。每次赋值给变量的是一个值，所以需要循环
		%不能直接将一列值赋给变量
		fetch 游标名称 into 变量 [, 变量];
		%关闭游标
		close 游标名;
```
	条件处理程序：
		定义：
			流程控制结构中执行过程遇到问题时相应的处理步骤，类似python的try except
		语法：
```
declare 处理方法 handler for 错误类型 SQL语句；

处理方法可选值：
	continue：继续执行
	exit：退出
错误类型可选值
	sqlstate sqlstate_value:
		某个具体的状态码，如02000,sqlstate_value指具体错误码，str类型
	sqlwarning：
		所有01开头的状态码
	not found：
		所有02开头的状态码
	sqlexception：
		所有其他状态码
SQL语句：
	相当于python的except后面的代码
```
游标代码示例：
```
%目标：将一个表中年龄大于uage的人的id和姓名和专业，插入新表
create procedure p11(in uage int)
begin
	declare uname varchar(10);	
	declare upro varchar(10);
	declare u_cursor cursor for select name,profession from tb_user where name <= uage;
	declare exit handler for sqlstate '02000' close u_cursor;
	
	drop if exists tb_user_pro;
	creat table if not exists tb_user_pro(
		id int primary key auto_increment,
		name varchar(10),
		proession varchar(10)
		);
	open cursor;
	while true do
		fetch u_cursor into uname,upro;
		insert into tb_user_pro value (null,uname,upro);
	end while;
	close cursor;
end
```
存储函数：
	定义：
		存储函数是有返回值的存储过程。存储函数的参数只能是in类型的
	语法：
```
create function 存储函数名 ([参数列表])
returns type characteristic
begin
	sql语句
	return
end;

returns指定返回值类型和特性
characteristic说明：
	deterministic：相同参数输入总有相同结果
	no sql:不包含sql语句
	reads sql data：包含读取数据语句，但不包含写入数据语句
```
	示例：
```
%目标：从1累加到n
creat function fun1(n)
returns int deterministic
begin
	declare tatal int default 0;
	
	while n>0 do
		set total := totale+n;
		set n:= n-1;
	end while;
	return total;
end;
```
